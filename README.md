## How to Run

### Clone the Repository

```bash
git clone https://github.com/hafizamehakarif/AI-Engineer-Internship-at-Xeven-Solutions.git
cd AI-Engineer-Internship-at-Xeven-Solutions
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run a Specific Day's Work

```bash
cd day_5
python filename.py
```

or open the corresponding notebook:

```bash
jupyter notebook
```

### Environment Variables

Some projects require API keys.

Create a `.env` file and store your credentials securely:

```env
OPENAI_API_KEY=your_key_here
GROQ_API_KEY=your_key_here
```

Never commit API keys to GitHub.
